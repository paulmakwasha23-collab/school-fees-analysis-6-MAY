# School Fees Analysis – Python Data Analytics Project

## Project Overview
This project analyzes simulated school fee payment data using Python (pandas, matplotlib, seaborn). It generates insights on total fees due, paid amounts, recovery rates, and payment status distribution across grades.

## Files in this Repository
| File/Folder | Description |
|-------------|-------------|
| `CodeTuesdaySchoolPythonDataAnalyst5May.py` | Main Python script |
| `client_deliverables/` | Output folder with all deliverables |
| `client_deliverables/01_payment_status.png` | Bar chart of payment statuses |
| `client_deliverables/01_total_due_by_grade.png` | Bar chart of total fees due by grade |
| `client_deliverables/Executive_Summary.txt` | Text summary with key metrics and insights |
| `client_deliverables/cleaned_student_fees.csv` | Cleaned dataset |
| `client_deliverables/summary_by_grade.csv` | Aggregated metrics by grade |

## Key Metrics Calculated
- **Total Fees Due** – sum of all FeesDue
- **Total Collected** – sum of all FeesPaid
- **Recovery Rate** – total collected / total due
- **Unpaid Accounts by Grade** – count of "Unpaid" status per grade
- **Average Balance** – average outstanding balance per student
